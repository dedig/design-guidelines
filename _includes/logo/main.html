Versões

Versão com byline

A versão com o byline é a preferencial, só deve ser substituída no caso das limitações descritas.

Versão simplificada

Usada em casos da leitura estar prejudicada. Como em dimensões reduzidas ou impressões com menor resolução.

Versão de uma cor

Para impressão em mídias limitadas a uma cor, caso não exista a limitação não usar essa versão.

Versão outline

Uso restrito para mídias onde o preenchimento deve ser evitado, caso não exista essa necessidade usar as outras versões.

Tamanho mínimo com byline

Impresso: 250 x 165 mm
Digital: 120 x 80 px

Tamanho mínimo sem byline

Impresso: 100 x 59 mm

Medida para margem

A margem deve ter largura da primeira letra "D", a margem deve ter a mesma espessura horizontalmente e verticalmente.
